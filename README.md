# ⚓ Ahoy — море общения

Социальная сеть в духе X (Twitter) с морским характером. Регистрация по email,
оформление профиля с фотографией и описанием, живая лента с постами,
изображениями и «якорями» вместо лайков.

## Возможности

- **Регистрация и вход по email** — с валидацией и хешированием пароля (SHA-256)
- **Онбординг профиля** — загрузка аватара и описание «о себе» сразу после регистрации
- **Лента постов** — публикация текста до 280 символов и изображений
- **Якоря (лайки)** — с анимацией всплеска
- **Редактирование профиля** — смена аватара и био прямо из ленты
- **Анимации повсюду** — framer-motion: переходы страниц, стаггер списков, микровзаимодействия
- **Стеклянный интерфейс** поверх живого фона морской волны

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Как это устроено

Всё хранение — на клиенте в `localStorage` (`src/lib/store.ts`): пользователи,
сессия и посты. Аватары и картинки постов сжимаются на канвасе до разумного
размера и хранятся как data URL. Это демо-архитектура: для продакшена
подключите настоящую БД и аутентификацию.

> ⚠️ Пароли хешируются SHA-256 на клиенте — этого достаточно для демо,
> но не для боевого использования.
